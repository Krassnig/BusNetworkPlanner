Bus Network Planner

The Bus Network Planner is a tool to help ciy planner plan an entire bus network from scratch. The program is target towards european cities. The process of creating a bus network is divided into four easy steps.

  1. Choose your city
  2. Adjust the city grid
  3. Select an algorithm
  4. See the result

Step 1. - Choose your city

Select any european city. I happen to be in Vienna, but want to edit Klagenfurt.

Step 2. - Adjust the city grid

Edit the street network of the city. Set intrest points such as the main train station. Change the population density.

Step 3. - Select an algorithm

Choose an algorithm and its parameters.

Step 4. - See the result

See the newly generated bus network und calculate efficient routes from A to B.
